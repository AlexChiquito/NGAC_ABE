# Test example
A minimal example of how to use pytest.

## Structure
The structure of the repository is as follows:
```bash
|- src/             # Source code
|  |- tests/        # Unit tests
|    |- test.py     # Test file
|  |- add.py        # Source file
|  |- main.py       # Main file
|- test.bash        # Bash script to run the tests
|- README.md        # This file
```
